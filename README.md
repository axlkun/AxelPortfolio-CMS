# UnfollowersTracker CMS

Fullstack Application developed with Vue - Laravel and Inertiajs - Jetstream where I can manage frontend sections of my portfolio: hero, about us, and contact, updating description and images. And I can create blog articles. All this data is exposed to an API to consume it from the frontend
